Arrays 

Data Structure 
        - a way to store, organize and manipulate data.
        - aren't tied to any one programming language
        - they can be implemented in many ways
        - exist in virtually any programming language

DS can be:
Linear
- Array (Dynamic or Static)
- Linked List
- Stack
- Queue

Non Linear
- Trees
- Graphs
- Tries

Single Linked List
    - is an additional linear data  structure that works a little bit different than an array
    - it's implementation is different and  allows us to have  some faster time complexity


Doubly Linked List
    - same with singly linked list exceept it has reverse pointer

Queue
    - is an additional linear data structure which allows us constant time operations to  access and remove elements from th front and the back of the queue
    - FIFO (First In First Out)

Binary Tree
    0  <- root (first node)      |
   / \ <- edges (line)           |
  0   0                          |   Height (the largest number of edges that exist in the trees)
 / \   \                         |   H of this tree is 2 because there is 2 edges
0   0   0 <- leaf (no child node)|   Depth of a node (How many edges have to go up in the tree to 
                                 |   reach the root node) also called the level of the node
Node Properties:
- Value
- Left
- Right
