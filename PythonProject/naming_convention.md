# Python Naming Conventions

## Quick Reference Table

| Code Element | Case Style | Example |
| :--- | :--- | :--- |
| **Variables** | `snake_case` | `user_age`, `total_score` |
| **Functions & Methods** | `snake_case` | `calculate_total()`, `get_data()` |
| **Classes** | `PascalCase` (CapWords) | `UserProfile`, `ShoppingCart` |
| **Constants** | `UPPER_SNAKE_CASE` | `MAX_LIMIT`, `PI` |
| **Modules (.py files)** | Short `snake_case` | `data_parser.py` |
| **Packages (Folders)** | Short lowercase (no underscores) | `analytics` |
